# p-net OrderReply Tool - 배포판

## 개요
p-net 시스템의 엑셀 파일 비교 및 수동 업로드 파일 생성 도구입니다.

## 설치 방법

### 1. Python 설치 확인
- Python 3.8 이상이 설치되어 있어야 합니다
- 설치 확인: 명령 프롬프트에서 `python --version`

### 2. 패키지 설치
압축 파일을 풀고, 폴더에서 다음 명령 실행:
```cmd
pip install -r requirements.txt
```

### 3. 실행
`run_app.bat` 더블클릭 또는 명령 프롬프트에서:
```cmd
python main.py
```

## 파일 설명

### 실행 파일
- `main.py` - GUI 메인 프로그램
- `excel_processor.py` - 엑셀 처리 로직
- `run_app.bat` - 간편 실행 배치 파일

### 설정 파일
- `requirements.txt` - 필요한 Python 패키지 목록

### 샘플 파일
- `sample_factory_reply.xlsx` - 공장납기회신 샘플 데이터

### 문서
- `README.md` - 상세 사용 설명서

## 사용 방법

1. **p-net 다운로드 파일 선택**: p-net에서 다운로드한 엑셀 파일
2. **공장납기회신 파일 선택**: 공장에서 받은 납기회신 엑셀 파일
3. **저장 경로 지정**: 결과 파일을 저장할 위치
4. **"파일 처리 및 생성" 버튼 클릭**

## 결과 파일 형식
생성되는 수동 업로드 파일의 컬럼 순서:
1. PO#
2. PO-LINE#
3. Material
4. CPO QTY
5. ETD
6. EX-F
7. 내부노트
8. CPO#
9. CPO-LINE#
10. LINE SEQ

## 시스템 요구사항
- Windows 10/11
- Python 3.8+
- pandas
- openpyxl
- tkinter (Python 표준 라이브러리)

## 문제 해결

### GUI가 실행되지 않는 경우
1. Python이 PATH에 등록되어 있는지 확인
2. 필요한 패키지가 설치되었는지 확인:
   ```cmd
   python -c "import pandas, openpyxl, tkinter"
   ```

### 파일 처리 오류
- 엑셀 파일 형식이 맞는지 확인
- 필수 컬럼이 있는지 확인
- 파일이 다른 프로그램에서 열려있지 않은지 확인
